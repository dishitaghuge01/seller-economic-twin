# Seller Economic Twin

Seller Economic Twin is a full-stack demo for exploring pricing, inventory, and forecasting scenarios through a seller-facing chat experience.

## Structure

- docs/: contracts and reports
- backend/: Python services and supporting data
- frontend/: Vite + React + Tailwind + Recharts UI

## Development

From the frontend directory:

```bash
npm install
npm run dev
```

## Build and test

```bash
npm run build
npm run test
```
